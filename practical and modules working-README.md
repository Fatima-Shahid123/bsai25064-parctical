# Glass Rod Module

## Overview
This module demonstrates the interaction and movement of a laboratory glass rod used for stirring solutions during experiments.

## Controls
- Click on the Glass Rod in Play Mode to activate it.
- Use the Arrow Keys to move the rod:
  - ↑ Up
  - ↓ Down
  - ← Left
  - → Right

## Stirring Feature
- Press the **O** key to rotate the glass rod in a circular motion.
- This movement simulates the stirring process commonly performed in laboratory experiments.
- The circular animation helps visualize how liquids are mixed using a glass rod.

## Learning Objective
This module demonstrates basic laboratory stirring techniques and object manipulation within a virtual lab environment.

# Test Tube Module

## Overview
This module simulates the handling and transfer of liquids using laboratory test tubes.

## Controls
### Test Tube Movement
- Click on the filled test tube to activate it.
- Use the following keys to move the test tube:
  - W → Forward
  - S → Backward
  - A → Left
  - D → Right

### Camera Controls
A dedicated Camera Controller script has been implemented to allow users to:
- Navigate around the laboratory environment.
- View the experiment from different angles.
- Easily position the test tube near target containers.

### Rotation Controls
- Press **Q** to rotate left.
- Press **R** to rotate right.

## Liquid Transfer
When the filled test tube is brought close to an empty test tube:
1. Rotate the filled test tube using Q or R.
2. The liquid can then be poured into the empty test tube.
3. This simulates real-world liquid transfer between laboratory containers.

## Learning Objective
This module demonstrates object movement, camera navigation, and realistic liquid transfer mechanics.


# Practical: Mixing and Stirring of Liquids

## Overview
This practical simulates a laboratory experiment involving the mixing and stirring of different liquids.

## Experiment Setup
- Six beakers are placed on the laboratory table.
- An empty beaker is positioned in the center and serves as the main mixing container.
- A special beaker containing Oil and Water is included to demonstrate the behavior of immiscible liquids.
- The oil layer is represented using a separate mesh layer to provide a realistic visualization during stirring.

## Available Liquids
The following liquids are provided:
- Water
- Ethanol

Each liquid has its own dedicated beaker.

## Procedure
1. Pick up the required liquid container.
2. Pour the liquids into the empty beaker located in the center.
3. Continue adding the required liquids according to the experiment.
4. Once the beaker contains the liquids, use the stirring functionality to mix them.

## Stirring Controls
- Hold the filled beaker.
- Press **S** to activate stirring.
- Observe the behavior of the liquids during the mixing process.

## Features Demonstrated
- Liquid pouring mechanics
- Container interaction
- Laboratory object manipulation
- Stirring simulation
- Oil and water layer visualization
- Experimental observation of liquid mixing

## Learning Objective
This practical provides a virtual representation of laboratory mixing procedures and helps users understand liquid interaction and stirring processes in a safe digital environment.
